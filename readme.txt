Quick View Woo adds a flexible Quick View button on your WooCommerce pages!

== Description ==
CSSIgniter Quick View for WooCommerce is the easiest way to improve your conversion rates on your ecommerce website. Your customers will be able to preview products and add them in their cart through the lightbox.

### Live Demo
Visit our [live demo](https://www.cssigniter.com/preview/quick-view-pro/).

### Documentation
Documentation is [available here](https://www.cssigniter.com/docs/quick-view-woo-pro/).

### Features
1. In your product listing pages you can add the Quick View button in the following locations:
	- Before Thumbnail
	- Before Title
	- Before Rating
	- Before Price
	- Before Add to Cart button
	- After Add to Cart button
2. Customize the “Quick View” button text.
3. Add custom CSS classes for easier customization
4. Toggle visibility of the “Quick View” button in mobile devices.
5. Toggle visibility of the “Quick View” button in related products, up-sells etc.
4. Change the width of the lightbox.
5. Change the colors of the lightbox content (Background, text & link colors, button colors and more!)

### Additional features can be found in our [Premium Add-on](https://www.cssigniter.com/plugins/quick-view-woo-pro/)

1. Include visibility of the “Quick View” button in certain categories only
2. Exclude visibility of the “Quick View” button from certain categories.
3. Customize the colors of the “Quick View” button.
4. Change the width of the image(s) area and the content area within the lightbox.
5. Toggle visiblity of the following lightbox product elements:
	1. Sales badge
	2. Image
	3. Gallery
	4. Title
	5. Rating
	6. Price
	7. Short Description
	8. Add to Cart button
	9. Meta description

Learn more about the [PRO Add-on here](https://www.cssigniter.com/plugins/quick-view-woo-pro/).

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/quick-view-woo` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the "Plugins" screen in WordPress
3. Visit your WooCommerce shop page and admire the new Quick View button!
4. Navigate to WooCommerce > Settings > Quick View, to customize your button and modal.
